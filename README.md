# Ab Timer

A small installable PWA for timed ab workouts.

## Try It Locally

Serve the folder from a local web server:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Put It On Your Phone

Deploy the folder to a static host such as Netlify, Vercel, or GitHub Pages. The PWA install and offline cache require an HTTPS URL on a phone.

After opening the hosted URL:

- iPhone: tap Share, then Add to Home Screen.
- Android: open the browser menu, then Add to Home screen or Install app.

The timer settings are saved in the browser on each device.
